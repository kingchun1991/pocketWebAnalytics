This is a simple blog post templates powered by Next.js, Chakra UI, TypeScript, and MDX. You can use this template to create your own blog posts.

Powered by [nextarter-chakra](https://github.com/sozonome/nextarter-chakra) from [sozonome](https://github.com/sozonome), this template created much simpler.

# Function & Features

1. 🤖 Search Engine Optimization (SEO) friendly: [Out-of-the-box nextjs 13 SEO](https://nextjs.org/blog/next-13-2#built-in-seo-support-with-new-metadata-api)
2. 🔍Search functionality.
3. 📚 MDX support

- ✅ [Basic Markdown to Chakra UI components](/blog/basic-markdown-components)
- 🕰️ [Custom components to Chakra UI components](/blog/custom-components)
- 🕰️ HTML to Chakra UI components

4. 🌈 Chakra UI for styling
5. 🌄 Automatic cover photo and og iamge by nextjs Open graph image generater

## Pre-requisites

1. [Node.js](https://nodejs.org/en/) or nvm installed.
2. `pnpm` installed.

# How to use this template in your local environemnt

1. [Clone this repository](https://github.com/new?template_name=nextjs-chakra-starter-blog&template_owner=kingchun1991)
2. cp .env.local.example .env
3. pnpm Install
4. pnpm dev
